# Setup
1. Node.js installieren [Linux](https://github.com/d0dge-dev/node-installer) oder für [Windows](https://nodejs.org/dist/v20.9.0/node-v20.9.0-x64.msi)
2. Navigiere in den Ordner wo die main.js Datei drin ist
3. Führe `npm i` aus um alle Packete zu installieren
4. Führe `chmod +x start.sh` aus (Linux only)
5. Starte bei Linux den Bot mit `./start.sh`
6. Starte bei Windows de Bot mit einem Doppelklick auf die start.bat

# Troubleshoot
- Config richtig ausgefüllt?
- Intents aktiviert beim Discord Bot?
- Token richtig?
- Node JS richtig installiert? -> Führe `node -v` aus, wenn kommt command nicht gefunden ist node js nicht richtig installiert
